NavGenerator V0.0.1

options

All the global options available.

Property Default Type Description
title / String The title of the navbar.
navItems(required) [] Array[Item] This is the array to store the nav items.
style / String This is custom css applied to the navbar.

options.navItems

All the information available for each nav item.

Type Description
Item You can set the nav items in two ways:
1. The simple way, you can only set one level to the navbar, e.g. 
[{
     name: "intro",
     url: "/intro"
},
{
     name: "buy",
     url: "/buy"
}]
2. The complex way, you can set first level with the child items to the navbar, e.g. 
[{
     name: "Home",
     url: "/",
     children: [{
         name: "intro",
         url: "/intro"
     },
     {
         name: "buy",
         url: "/buy"
     }]
},
{
     name: "About",
     url: "/about",
}]

Methods

Methods to control the navbar.

/**
  * to add new nav item
  *
  * @param name {string} example: "Home"
  * @param url {string} example: https://www.google.com
  * @param children {array} example: [{name: "", url: ""}]
  * @returns {NavGenerator}
  */
NavGeneratorInstance.addBar(name, url, children);

/**
  * set the navbar vertical
  *
  */
NavGeneratorInstance.setVertical();

/**
  * set the navbar horizontal
  *
  */
NavGeneratorInstance.setHorizontal();

/**
  * set the menu line style
  *
  *  @param {string} style example: "solid" or "dotted" or "dashed" or "none"
  */
NavGeneratorInstance.setLineStyle(style);

/**
  * set the navbar background color
  *
  * @param color {string} example: #fff
  * @returns {NavGenerator}
  */
NavGeneratorInstance.setBackgroundColor(color);

/**
  * set the navbar title
  *
  * @param title {string} example: "Title"
  * @returns {NavGenerator}
  */
NavGeneratorInstance.setTitle(title);


/**
  * delete the item from the navbar
  *
  * @param {string} name
  * @returns {NavGenerator}
  */
NavGeneratorInstance.deleteItem(name);

/**
  * set the menu background color
  *
  * @param {string} color example: #fff
  * @returns {NavGenerator}
  */
NavGeneratorInstance.setMenuBackgroundColor(color);

/**
  * set the menu border radius
  *
  * @param {string} radius  example: "5px"
  * @returns {NavGenerator}
  */
NavGeneratorInstance.setMenuBorderRadius(radius);

/**
  * get the navbar item list
  *
  * @returns {Array}
  */
NavGeneratorInstance.getNavItems();

/**
  * remove the title
  *
  * @returns {NavGenerator}
  */
NavGeneratorInstance.removeTitle();

/**
  * add a new menu item to the navbar item list
  *
  * @param targetName {string}
  * @param name {string}
  * @param url {string}
  * @returns {NavGenerator}
  */
NavGeneratorInstance.addItem(name, url, children);


/**
  * set the item color by name
  *
  * @param name {string} example: "Home"
  * @param color {string} example: #fff
  */
NavGeneratorInstance.setItemColorByName(name, color);

/**
  * set all the navbar item color
  *
  * @param color {string} example: #fff
  */
NavGeneratorInstance.setNavItemColor(color);


/**
  * set all the menu item color
  *
  * @param color {string} example: #fff
  */
NavGeneratorInstance.setMenuItemColor(color);


/**
  * remove the item from the navbar item list
  *
  * @param name {string}
  * @returns {NavGenerator}
  */
NavGeneratorInstance.removeItem(name);

Complete configuration demo

First, you should include the javascript file in the html.

 <script defer type="text/javascript" src='js/library.js'></script>
 <link type="text/css" rel="stylesheet" href="css/library.css">

Second, you should create a html element with an id.

 <div id="navbar1"></div>

Complete configuration with two levels, you can test it in pub/js/example.js

const ng = new NavGenerator("navbar1", {
    navItems: [
        {
            name: "Home",
            url: "/",
            children: [
                {
                    name: "intro",
                    url: "/intro"
                },
                {
                    name: "buy",
                    url: "/buy"
                },
                {
                    name: "sell",
                    url: "/sell"
                },
                {
                    name: "rent",
                    url: "/rent"
                }
            ]
        },
        {
            name: "About",
            url: "/about",
        },
        {
            name: "Contact",
            url: "/contact",
            children: [
                {
                    name: "email",
                    url: "/email"
                },
                {
                    name: "phone",
                    url: "/phone"
                },
                {
                    name: "address",
                    url: "/address"
                }
            ]
        }
    ],
    title: "Nav Generator",
});

// add a bar to the navbar
ng.addBar("hhhh", "/asd", [{
    name: "asd",
    url: "/asd"
}]);

// set the navbar background color
ng.setBackgroundColor("#ffffee");

// set the menu border radius
ng.setMenuBorderRadius("10px");

// delete a item from the navbar
ng.deleteItem("phone");

// add a menu item to the navbar
ng.addMenuItem("Contact","twitter", "/twitter");

// set the menu title
ng.setTitle("Test Tittle");


// set the menu background color
ng.setMenuBackgroundColor("#feb373");

// set the nav item font color
ng.setNavItemColor("green");

// set the menu item font color
ng.setMenuItemColor("blue");

// set a item font color
ng.setItemColorByName("address", "red");

// remove the item from the navbar
ng.removeItem("Contact");

// set the navbar vertical
ng.setVertical();

// set the menu line style
ng.setMenuLineStyle("dashed");
